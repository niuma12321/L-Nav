```

│                          用户浏览器 (Client)                               │
├─────────────────────────────────────────────────────────────────────────┤
│  localStorage:                                                           │

│  ├── faviconCache (图标缓存)                                            │

│                                                                          │
│  Memory (React State):                                                   │
│  ├── 当前编辑状态                                                        │
│  ├── 临时搜索数据                                                        │
│  └── UI 状态                                                             │
└─────────────────────────────────────────────────────────────────────────┘
                                    │
                                    │ HTTPS / API
                                    ▼
┌─────────────────────────────────────────────────────────────────────────┐
│                    Cloudflare Workers (Edge)                               │
├─────────────────────────────────────────────────────────────────────────┤
│  API 路由处理:                                                           │
│  ├── /api/v1/sync    → 数据同步                                          │
│  ├── /api/v1/links   → 链接管理                                        │

│                                                                          │

│  ├── 结构化数据 → D1 数据库                                              │
│  ├── 大文件数据 → R2 存储桶                                               │
│  └── 缓存/临时 → KV 存储                                                 │
└─────────────────────────────────────────────────────────────────────────┘
                                    │
                    ┌───────────────┼───────────────┐
                    │               │               │
                    ▼               ▼               ▼
┌─────────────────────────┐ ┌──────────────┐ ┌──────────────────────┐
│      D1 数据库           │ │  R2 存储桶    │ │    KV 存储           │
├─────────────────────────┤ ├──────────────┤ ├──────────────────────┤
│  links 表               │ │              │ │                      │
│  ├── 链接标题           │ │ 备份文件:    │ │  sync:data           │

│  ├── 固定/隐藏状态      │ │              │ │                      │
│  └── 排序索引           │ │ 图标缓存:    │ │  static:*            │
│                         │ │  ├── 网站    │ │  (静态资源)           │
│  categories 表          │ │  │   favicon │ │                      │
│  ├── 分类名称           │ │  └── 导出    │ │  session:*           │
│  ├── 图标               │ │      图片    │ │  (临时会话)          │
│  └── 排序               │ │              │ │                      │
│                         │ │ 用户上传:    │ │  rate_limit:*        │
│  user_settings 表       │ │  └── 自定义  │ │  (限流计数)         │
│  ├── 主题               │ │      头像等  │ │                      │
│  ├── 同步密码哈希       │ │              │ │                      │
│  ├── AI配置 (JSON)      │ │              │ │                      │
│  └── 站点设置 (JSON)    │ │              │ │                      │
│                         │ │              │ │                      │

│  ├── 名称               │ │              │ │                      │
```

## 数据类型映射表

| 数据类型 | 存储位置 | 原因 |
|---------|---------|------|
| **链接/分类** | D1 + KV 备份 | 结构化数据，需要查询和排序 |
| **用户设置** | D1 | 小体积结构化数据，频繁读取 |
| **完整备份** | R2 + D1 元数据 | 大文件，D1 存元数据便于列表查询 |
| **图标缓存** | R2 | 文件存储，减少重复下载 |
| **同步状态** | KV | 快速读写，边缘缓存 |
| **临时会话** | KV | 短期数据，自动过期 |
| **限流计数** | KV | 原子操作，快速 |
| **静态资源** | KV (Workers Assets) | CDN 缓存，全球分发 |

## 存储特性对比

```
┌─────────┬────────────┬──────────┬────────────┬──────────┐
│ 存储    │ 数据类型   │ 读取速度 │ 写入速度   │ 适用场景 │
├─────────┼────────────┼──────────┼────────────┼──────────┤
│ D1      │ SQL 结构化 │ ⭐⭐⭐⭐  │ ⭐⭐⭐      │ 主数据   │

└─────────┴────────────┴──────────┴────────────┴──────────┘
```

## API 数据流向示例

### 1. 保存链接
```
浏览器 → POST /api/v1/links
              ↓
         Worker 处理
              ↓
    ┌─────────┴─────────┐
    │                   │
    ▼                   ▼ (失败回退)
   D1: links 表        KV: sync:data
```

### 2. 创建备份
```
浏览器 → POST /api/v1/backup
              ↓
         Worker 处理
              ↓
    ┌─────────┴─────────┐
    │                   │
    ▼                   ▼
   R2: 备份文件        D1: backups 表
   (backups/*.json)   (元数据)
```

### 3. 同步数据
```
浏览器 → GET /api/v1/sync
              ↓
         Worker 处理
              ↓
    ┌─────────┴─────────┐
    │                   │
    ▼                   ▼ (D1 失败)
   D1: 查询所有表      KV: sync:data
   links/categories    (完整 JSON)
   合并返回
```

## 容错机制

```
D1 失败 → 自动回退到 KV
R2 失败 → 使用 D1 BLOB (小文件) 或返回错误
KV 失败 → 仅使用 D1 (读模式)
```

## 部署配置

wrangler.toml 中的存储绑定：
- `YNAV_D1`: D1 数据库 (结构化数据)
- `YNAV_R2`: R2 存储桶 (大文件)
- `YNAV_WORKER_KV`: KV 命名空间 (缓存)
